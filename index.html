Guía básica para desplegar una página web utilizando Cloudflare Pages.

🚀 Paso a Paso

  • Crear el archivo HTML.
  • Crear un repositorio en GitHub.
  • Subir el proyecto a GitHub.
  • Crear una cuenta en Cloudflare.
  • Conectar Cloudflare con GitHub.
  • Configurar el despliegue.
  • Publicar la página.
  • Verificar el resultado.

📋 Requisitos

  1. Cuenta en GitHub.
  2. Cuenta en Cloudflare.
  3. Archivo HTML.
  4. Conexión a Internet.
  5. Navegador web.
